
# Code of Conduct

This project is dedicated to providing a welcoming, respectful, and inclusive environment for everyone. All contributors, maintainers, and community members are expected to follow these guidelines.

## Our Standards

- Be respectful and considerate in language and actions.
- Assume good intentions and be open to discussion.
- Provide constructive, actionable feedback.
- Respect privacy and do not share private information.

## Unacceptable Behavior

Examples of unacceptable behavior include, but are not limited to:

- Harassment, hateful or discriminatory language or imagery.
- Threats, stalking, or sustained disruption of project activities.
- Sharing private or personally identifying information without consent.
- Sexual language, unwanted sexual attention, or sexual images in public spaces.

## Scope

This Code of Conduct applies to all project spaces, including issue trackers, pull requests, mailing lists, chat channels, and in-person events organized by the project.

## Reporting

If you experience or witness unacceptable behavior, please report it to the maintainers. Include as much information as possible (what happened, when, and any relevant links or screenshots). Reports will be handled confidentially to the extent possible.

If the repository `README.md` contains a preferred contact, use that; otherwise open an issue or contact the maintainers directly via the project's listed channels.

## Enforcement

Maintainers will review reports and take appropriate actions, which may include a private warning, temporary removal from project spaces, or a permanent ban. The maintainers' decisions are final.

## Acknowledgements

This Code of Conduct is intended to be short and practical. Projects may adopt more detailed policies as needed.
