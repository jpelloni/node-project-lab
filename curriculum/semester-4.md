# Semester 4 – Developer Tooling

## Mission

Developer productivity tools shape how software is written.

This semester shifts focus from application development to the tools that developers use every day. We'll explore compiler APIs, Abstract Syntax Trees (ASTs), package management, editor extensions, static analysis, and command-line applications.

Rather than consuming existing tooling, we'll build our own.

## Learning Objectives

* Learn the TypeScript Compiler API.
* Explore Abstract Syntax Trees.
* Build command-line applications.
* Develop custom ESLint rules.
* Create codemods and automated refactoring tools.
* Build a Visual Studio Code extension.
* Package and distribute reusable developer tools.

## Capstone

Build a polished developer productivity tool that solves a real-world engineering problem and can be shared with the wider community.
