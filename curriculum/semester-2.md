# Semester 2 – Networking and Communication

## Mission

Software becomes valuable when it communicates.

This semester explores how Node.js interacts with the outside world by building networking components from first principles. Rather than relying on frameworks, we'll begin with raw TCP sockets and progressively work upward through HTTP, TLS, WebSockets, and application protocols.

By constructing these components ourselves, we'll gain a deeper appreciation for the abstractions provided by frameworks like Express, Fastify, and NestJS.

## Learning Objectives

* Understand TCP/IP fundamentals.
* Build HTTP servers from the native Node APIs.
* Learn how requests and responses are processed.
* Explore DNS, TLS, and secure communication.
* Build networking utilities using Streams.
* Understand how popular web frameworks build upon Node's core modules.

## Capstone

Build a lightweight web framework that demonstrates routing, middleware, and request handling using only Node's built-in capabilities.
